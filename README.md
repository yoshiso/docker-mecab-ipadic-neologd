# Dockernized mecab-ipadic-neologd

## Requirement

Docker

## Install

```
$ git clone https://github.com/yss44/docker-mecab-ipadic-neologd
$ cd docker-mecab-ipadic-neologd
$ chmod +x build run
$ ./build
```
